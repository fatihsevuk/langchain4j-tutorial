
# Java LLM CLI Interface with Langchain4j

## Overview

This Java application provides a command line interface (CLI) for interacting with large language models (LLMs) using the Langchain4j library. The application allows users to input text prompts and receive responses generated by the LLM.

## Features

- Easy-to-use CLI for interacting with LLMs
- Supports customizable text prompts
- Built on Langchain4j for efficient language model interaction
- Simple setup and configuration
